像素化表格与响应式图片实战

2026-06-30

在 Minecraft 风格的网页中，表格和图片同样需要遵循像素化、块状的设计语言。本文将展示如何使用纯 CSS 让表格和图片融入方块世界，同时保持良好的可读性与响应式表现。

像素风格表格设计

表格必须保留直角、双线边框和立体阴影，与卡片风格一致。表头使用深色背景与大写字母，数据行交替时不做额外装饰，保持硬朗感。

基础表格结构

使用 <table> 标签，搭配 border-collapse: separate 以保留双线边框。示例表格展示了不同方块的材料属性：

方块 硬度 爆炸抗性 可再生
草方块 0.6 0.6 是
石头 1.5 6.0 否
黑曜石 50.0 1200.0 否
木板 2.0 3.0 是

表格的 CSS 实现

核心样式如下，通过 box-shadow 和 inset 模拟立体边框，无圆角，字体使用 Mojangles。

.mc-table {
  width: 100%;
  border: 3px solid #2d2d2d;
  border-collapse: separate;
  border-spacing: 0;
  background: #1a1a1a;
  box-shadow:
    inset 2px 2px 0 rgba(0,0,0,0.4),
    3px 3px 0 rgba(0,0,0,0.5);
}
.mc-table th {
  background: #2A2A2A;
  font-family: 'Mojangles', sans-serif;
  text-transform: uppercase;
}
.mc-table td {
  border: 2px solid #2d2d2d;
  background: #1E1E1E;
}

注意：表格边框使用 separate 模式是为了保留单元格之间的间隙，这样才能实现双线效果。若使用 collapse，边框会合并成一条，失去像素感。

响应式图片适配

图片在 Minecraft 风格中需要像素化边框，并且根据横竖方向自适应。横版图片占满内容宽度，竖版图片限制最大高度并居中。

竖版图片会按比例缩小，最大高度不超过视口高度的 70%，保持居中。

图片 CSS 关键规则

通过 max-width: 100% 和 max-height: 70vh 实现自适应，同时赋予像素化边框。

img {
  max-width: 100%;
  max-height: 70vh;
  width: auto;
  height: auto;
  display: block;
  margin: 1.5em auto;
  border: 3px solid #2d2d2d;
  box-shadow: 4px 4px 0 rgba(0,0,0,0.5);
  image-rendering: pixelated;
}

以上就是表格和图片在像素风博客中的实现方法。将这套样式应用到实际项目中，你的页面就能轻松融入 Minecraft 的美学体系。
